README for Lottie Animations
=============================

Bu klasör Lottie animasyon dosyalarını içerir.

Şu anda basit JSON animasyonlar kullanılıyor.
Daha profesyonel animasyonlar için:

1. LottieFiles (https://lottiefiles.com/) üzerinden ücretsiz animasyonlar indirebilirsiniz
2. Adobe After Effects ile özel animasyonlar oluşturabilirsiniz
3. Bodymovin plugin ile AE projelerini Lottie JSON'a çevirebilirsiniz

Önerilen animasyonlar:
- success.json ✅ (mevcut)
- error.json ❌ 
- loading.json ⏳
- celebration.json 🎉
- empty.json 📭
